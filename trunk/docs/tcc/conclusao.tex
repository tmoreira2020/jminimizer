\chapter{Conclusão}
O desenvolvimento de aplicações baseadas em bibliotecas de terceiros é ivariavelmente uma solução inteligente quando não queremos ou podemos desenvolver infra estrutura para nossa aplicação. Mas o impacto do tamanho da biblioteca no tamanho final da aplicação pode inviabilizar o projeto. A tese aqui desenvolvida criou uma ferramenta capaz de minimizar o impacto do uso de bibliotecas. Analisando o código estaticamente e removendo estruturas inúteis a aplicação oriundas dessas bibliotecas. Além disso essa ferramenta foi desenvolvida visando ser um projeto open-source, e atualmente está disponível em https://jminimizer.dev.java.net/.
