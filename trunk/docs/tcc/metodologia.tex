\section{Conceitos}
Segundo Demo\cite{demo} diz "Metodologia é a preocupação instrumental. Trata das formas de se fazer ciência. Cuida dos procedimentos, das ferramentas, dos caminhos".

Já Andrade\cite{andrade} afirma "Métodos são técnicas suficientemente gerais para se tornarem comuns a todas as ciências ou a uma significativa parte dela".

Um método serve para descobrir o problema, encontrar soluções e validá-las\cite{lakatos}.
